# Small-Talk-AI-Agent--Ashta
Ashta is an AI-powered conversational agent designed to enhance small talk in everyday social interactions.

## How to replicate the AI Chatbot:
1. Clone the repo using the command "git clone https://github.com/AvinashReddyG/Small-Talk-AI-Agent--Ashta"
2. Open the folder on VS Code or Code Editor and replace the 'YOUR_API_KEY' in .env file with Gemini API Key.
3. Open the Terminal and run the following commands:
   npm install
   npm run dev
4. You will get the link to open on your browser using local port. Press Ctrl and click on the link to open the webpage.
5. You can now start using the AI chatbot by clicking on chat icon at the right bottom of the page.
6. Enter the prompt you want and click Send(Arrow icon).

## Some features to explore:
1. You can change the theme by clicking on the moon icon on the webpage.
2. You can turn on or off the 'Sound' icon to hear/mute the output message in the chatbot. By default, the sound will be     turned off. You can use this to learn pronounciations of complex words.
3. You can reset the conversation by clicking on the Reset icon next to the name of chatbot.
4. You can minimize the chatbot by clicking on the Down arrow button next to Reset icon.
5. There is 'Quick Options' button next to Sound icon for some quick contexts.
6. After the output is generated, there will be 'Refresh' button next to the message box. By clciking on it, you wil get different responses for the above prompt.
